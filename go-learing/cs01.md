# GO 基础

## 函数

### 关键字 
 
`func`

### 定义方式

#### 有返回值

```golang
func add (x ,y int) int {
  return x + y
}

```


#### 无返回值

```golang
func printObject(str string){
  fmt.println(str)
}
```

## 数组

#### 定义方式

```golang
var varName = [] type {initItems}
//如
var array = [] int {1,2,3,4}
//简写
array := [] int {1,2,3,4}
```

## 结构体
`相当于C#中的类`

### 关键字
`struct`

### 定义方式
```golang
type struct_variable_type struct {
   member definition;
   member definition;
   ...
   member definition;
}
```

### 实例如下
```golang
//定义
type Book struct {
	title  string
	author string
}
//创建一个结构体
fmt.println(Book{title:"C语言学习",authr:"李四"})
//使用
var Book1 Book        /* 声明 Book1 为 Book 类型 */
var Book2 Book        /* 声明 Book2 为 Book 类型 */
//赋值
Book1.title = "这是一本书"
Book1.author = "张三"

```
